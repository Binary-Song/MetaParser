输入文件解析器¶

class InputFileResolver¶
#include <input-resolver.hpp>

输入文件解析器。 InputFileResolver 能将以规定格式写成的文件解析成 Rules 对象。调用 resolve_input_file 来解析文件，每个对象只能调用一次这个函数。

Public Functions

InputFileResolver()¶

构造涉及正则表达式的编译，较耗时，应谨慎拷贝本类对象。

int resolve_input_file(const char *file_name, Rules &rules)¶

读取输入文件，输出各种规则(放在 rules 中)。如果发生错误，返回错误数量（0为成功），诊断信息存放在full_diag_msg中。每个对象只能调用一次。

Parameters

  • file_name – 输入文件的路径

  • rules – [out] 输出的规则集

Returns

错误数量（0为成功）
